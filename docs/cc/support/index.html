CadetCatch

Support

Use these notes for common CadetCatch setup steps, troubleshooting, and App Review follow-up.

Core setup

  • Activate CadetCatch Pro from the first-run paywall.
  • Add a cadet from the Roster tab and choose a base photo if available.
  • Open Radar, select a scan mode, and run a sweep.
  • Review matches in Intel, save important matches, and generate sitrep drafts.

Troubleshooting

  • If a selected roster photo does not appear, choose the photo again from Roster.
  • If match cards are empty, return to Radar and run a new sweep.
  • If purchase products are unavailable, confirm the App Store Connect subscription is active.
  • Notification settings can be managed in iOS Settings after permission is requested.

Contact

For support requests, bug reports, or review follow-up, contact https://eb28.co/#contact.
