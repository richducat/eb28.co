## Description

CadetCatch helps Coast Guard Academy families keep up with public cadet photos without digging through every gallery by hand.

Create a private roster on your iPhone, add approved public photo sources, scan those sources for possible face matches, review match confidence, save likely finds, and keep simple notes for family follow-up.

CadetCatch only checks public HTTPS sources that you add or enable. It does not scan private social accounts, private photo libraries, or non-public websites unless a future connected-source version explicitly asks for that access.
