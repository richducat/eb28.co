CadetCatch Support

CadetCatch is an iPhone app for cadet families to keep a private local roster and review possible matches from public photo sources.

Getting started

  1. Add a cadet in Roster with a clear profile photo selected from Photos.
  2. Open Sources and enable the public HTTPS pages you want to check.
  3. Open Home and run Check Public Sources.
  4. Review possible matches in Photos, save likely finds, and add review notes.

Troubleshooting

  • If there are no matches, confirm the source page contains public image links and run the check again.
  • If the app says no face was detected, choose a clearer front-facing profile photo.
  • If a website blocks image access or changes its page markup, CadetCatch will show an empty or no-match result.
