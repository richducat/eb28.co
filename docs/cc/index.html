Built for family photo review

CadetCatch keeps the workflow simple: add a cadet profile, choose public sources, run a check, and review possible matches with confidence scores.

Private roster

Cadet profiles and selected profile photos stay local on the device.

Public sources

The app checks public HTTPS pages that are built in or added by the user.

Review first

Possible matches can be saved or ignored. Empty and no-match results are shown plainly.

Privacy boundary

This build does not scan private social accounts, private photo libraries, private websites, or logged-in pages.
