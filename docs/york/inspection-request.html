Frequently asked questions

Before you request an inspection.

What if I choose the wrong inspection type?

No problem. York Inspections reviews each request and confirms the proper inspection before the appointment.

Do I pay through this form?

No. This request form does not collect payment. If you want to schedule or pay online, use the service link shown after you select an inspection.

What should I put in the notes?

Include the insurance deadline, closing date, agent instructions, gate code, lockbox details, roof age questions, or any special access instructions.

Where does York Inspections work?

York Inspections is based in Merritt Island and serves Brevard County, including Cocoa, Rockledge, Viera, Melbourne, Titusville, Cape Canaveral, and Palm Bay.
