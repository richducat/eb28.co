## Daily workflow (15 min)
1. Open `calendar-30day.md`, find today's row.
2. Copy the matching post from the platform file.
3. Post it. On LinkedIn/Reddit put links in the first comment, not the body (reach + anti-spam).
4. Reply to every comment for the first 2 hours — the thread is the demo.
5. Need more? Run `node scripts/deskos-social-engine.mjs --count 10`.
