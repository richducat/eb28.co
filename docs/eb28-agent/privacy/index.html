Eb28 Agent Privacy Policy

This Privacy Policy explains how Eb28 Agent handles information when authorized users connect Google Workspace accounts to EB28 assistant automation workflows.

Information we access

Depending on the permissions granted by the user, Eb28 Agent may access Google account information such as email metadata and message content, calendar events, Drive files, Docs, Sheets, contacts, and task-related information. The exact access depends on the OAuth scopes approved by the user.

How we use information

We use Google user data only to provide assistant functions requested or configured by the authorized user, including calendar briefs, inbox triage, document review, meeting preparation, follow-up tracking, and related administrative automation.

Data sharing

We do not sell Google user data. We do not share Google user data with third parties for advertising. Information may be processed by infrastructure providers solely as needed to run the app and provide the requested assistant functionality.

Data storage and retention

Assistant outputs, task notes, summaries, and operational logs may be stored in the user's private assistant workspace for continuity and follow-through. Data is retained only as long as needed for the user's assistant workflows or legal/security requirements.

Google API Services User Data Policy

Eb28 Agent's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including the Limited Use requirements.

User control and deletion

Users can revoke access at any time from their Google Account permissions page. To request deletion of assistant-stored data associated with Eb28 Agent, contact us at richard@eb28.co.

Contact

Questions about this policy can be sent to richard@eb28.co.
