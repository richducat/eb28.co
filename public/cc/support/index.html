Need help with CadetCatch, public photo sources, saved finds, or purchases? Email support@eb28.co.

Common Help

  • Add a cadet from Roster using a clear face photo selected from Photos.
  • Use Sources to enable approved public photo pages or add a secure page you can open in Safari.
  • Use Home > Check Photos to scan selected public sources for possible matches.
  • Use Restore Purchases from Photo Access or More if you already purchased access.
