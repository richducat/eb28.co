Eb28 Agent

Eb28 Agent is a private assistant automation app used by authorized users to connect Google Workspace data to internal executive-assistant workflows, calendar briefs, inbox triage, and follow-up tracking.

Privacy Policy Contact

What the app does

Eb28 Agent helps approved users review their own Google Calendar, Gmail, Drive, Docs, Sheets, and related Workspace information so assistant workflows can surface upcoming meetings, action items, unread follow-ups, and operational blockers.

  • Creates calendar and meeting briefs for authorized accounts.
  • Summarizes inbox and document context when the user asks or has configured assistant monitoring.
  • Tracks follow-ups and reminders inside the user's assistant workspace.
  • Does not provide public self-serve access; accounts must be explicitly authorized.

Access and authorization

Eb28 Agent only accesses Google data after a user completes Google's OAuth consent flow. Access is limited to the scopes granted by the user and is used only to provide the assistant functions described here.

For privacy details, including data handling and deletion requests, read the Eb28 Agent Privacy Policy.
